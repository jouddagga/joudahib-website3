# AHIB – Al Hosson Insurance Brokers L.L.C

Professional insurance company website built with React + Vite + Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Build for Production

```bash
npm run build
npm run preview
```

## Stack

- React 18
- TypeScript
- Vite 6
- Tailwind CSS v4

## Features

- Fully responsive design
- Bilingual support (English / Arabic) with RTL
- Smooth scroll navigation
- Sections: Hero, About, Services, Why Us, Gallery, Contact
- Contact form
- Image lightbox gallery
